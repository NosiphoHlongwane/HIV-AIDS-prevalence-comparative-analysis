# HIV Prevalence Comparative Analysis

This project explores HIV prevalence, number of cases, and annual deaths across multiple countries.
Using Python (Pandas, Matplotlib, Seaborn), to analyse the global HIV/AIDS patterns to understand the relationship between prevalence, mortality, and access to healthcare.
The study highlights disparities in outcomes and shows how treatment access shapes HIV-related mortality.

## About the Project
The project focuses on comparing HIV prevalence, cases, and deaths across different countries.
It covers:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Correlation analysis between prevalence, cases, and deaths
- Visualisations showing healthcare effectiveness

## Dataset
The dataset includes:
- HIV prevalence rates (%)
- Number of HIV cases
- Annual HIV-related deaths

## Results
- High prevalence does not always equal high deaths — strong healthcare systems reduce mortality.
- Some countries with lower prevalence still face high mortality due to limited treatment access.
- Correlations show prevalence and deaths are related, but socio-economic and healthcare factors strongly influence outcomes.
